# Team01 — BeCode First Assignment

A simple team introduction website built with plain HTML, CSS, and JavaScript.

## 🗂️ Project Structure

```
Test01/
├── index.html       ← Main page
├── css/
│   └── style.css    ← All styles
├── js/
│   └── main.js      ← Interactions
└── README.md
```

## 👥 How to Add Yourself

1. Open `index.html`
2. Find the `<!-- ✏️ Add your card -->` comment
3. Copy an existing `<article class="card">` block
4. Update your **name**, **role**, **bio sentence**, and **GitHub link**
5. Pick a `--hue` value (0–360) for your card's accent color

## 🚀 Run Locally

No build step needed — just open `index.html` in your browser:

```bash
# Option 1: double-click index.html
# Option 2: use a live server
npx serve .
```

## ✨ Tech Stack

- HTML5
- CSS3 (custom properties, grid, animations)
- Vanilla JavaScript (IntersectionObserver)
